# Pomodoro App

Pomodoro App is a task manager app

## Technologies
- npx
- Angular v11
- Firebase
- Angular Material

## Using this repo
You are free to use this repo, just fork it

### To start dev server
Add your firebase credentials to firebase.ts
Then just run
```
    yarn install
    yarn start
```
## Build
```
    yarn build --prod
```
